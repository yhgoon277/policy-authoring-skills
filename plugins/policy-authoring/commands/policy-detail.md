# Policy Detail Authoring

Use this command when the user wants to write, rewrite, fact-check, table-format, or map policy detail items.

## Arguments

The user invoked this command with: $ARGUMENTS

## Instructions

1. Read `skills/policy-detail-authoring/SKILL.md`.
2. When needed, load `skills/policy-detail-authoring/references/pi-format.md` and `skills/policy-detail-authoring/references/factcheck-and-tobe.md`.
3. Produce PI content with readable rules, criteria, notices, evidence, optional tables, `field_review`, and `applies_to` mappings.
4. Keep to-be improvement candidates separate from the policy body.

## Example Usage

```text
/policy-detail 이 정책 상세를 표로 정리하고 현업 검토 필요 값을 표시해줘
/policy-authoring:policy-detail PI 초안을 가독형으로 다시 작성해줘
```
